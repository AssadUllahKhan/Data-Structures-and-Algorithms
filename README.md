# 📘 Data Structures and Algorithms

This repository contains implementations and concepts of **Data Structures and Algorithms (DSA)**, including sorting, searching, recursion, trees, graphs, and complexity analysis. It serves as a structured collection of coursework materials and practice problems to strengthen problem-solving skills.

---

## 📌 About This Repository

The purpose of this repository is to:

- Build strong fundamentals in Data Structures  
- Improve algorithmic thinking  
- Practice efficient and optimized coding  
- Maintain organized academic resources  

---

## 📚 Topics Covered

- Arrays and Strings  
- Linked Lists  
- Stacks and Queues  
- Recursion  
- Sorting Algorithms  
  - Bubble Sort  
  - Selection Sort  
  - Insertion Sort  
  - Merge Sort  
  - Quick Sort  
- Searching Algorithms  
  - Linear Search  
  - Binary Search  
- Trees (Binary Tree, BST, Traversals)  
- Graphs (BFS, DFS)  
- Hashing  
- Time and Space Complexity Analysis  

---

## 💻 Programming Languages

- C / C++
